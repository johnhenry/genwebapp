This site is a good starting point for web applications.

You'll notice a number of things built in:

  • FrontEnd
  • Angular as a general framework.
  • Bootstap + Sass as a for styles and widgets.
  • React for smaller components.
  • GSAP for animating on page elements. (It's responsible for the transitions on the site.)
  • Snap SVG for animating vector graphics.
  • jQuery a general utiltiy.
  • Lodash a general utiltiy.
  • d3 a data visuialzation framework.
  • CryptoJS some parts used for hashing (https://code.google.com/p/crypto-js/).
  • Backend
  • Grunt a task runner.
  • Yo for scaffolding.
  • Bower for package management.
  • Node runs a lot of stuff in teh background
  • Ruby/Compass for compiling sass

And here's what I plan on adding...

  • Three.js for rendering 3D assets.
  • Cheat.js adding extras
  • Something for WEBRTC, SOAP
  • Connections to backend services like FireBase, Parse
